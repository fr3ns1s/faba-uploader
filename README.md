# Faba uploader

Script python per la gestione delle playlist dei faba-me

## Prerequisiti

Prima di eseguire il progetto, assicurati di avere i seguenti requisiti installati:

- Python 3.x
- pip (Python package installer)

## Installazione
pip install -r requirements.txt

## Configurazione
